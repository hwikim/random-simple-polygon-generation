-------------------------------------------------------------------------------
                       README: Simple Polygon Generation
-------------------------------------------------------------------------------

Instructions:

1.  Install matplotlib (https://matplotlib.org/3.1.1/users/installing.html).

2.  Generate a random point set by running generate.py on terminal. The 
    algorithm will not run without the point set having been created first.

        python3 generate.py

    Answer the prompt ("Number of vertices: ") with an integer greater than 2.

3a. For the naive approach, run main_naive.py on terminal.

        python3 main_naive.py

3b. For the improved approach, run main_improved.py on terminal. 

        python3 main_improved.py
    
    Once the algorithm finishes running, a matplotlib plot with the final 
    polygon will display on a separate window. The terminal will display the
    number of vertices in the polygon and the execution time taken for the 
    polygon generation.
